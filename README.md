# 史地時光軸・台灣・中國・世界 三軌歷史對照

一個互動式的歷史學習網頁：把台灣、中國、世界三條歷史線以「同一個西元年份」並排對照，
從史前一路到 2017 年。可搜尋、可篩選、有上課模式、小測驗、歷史特輯，以及可唱誦記憶的朝代歌。

## 如何使用

直接用瀏覽器打開 `index.html` 即可，無需網路、無需安裝。打開後先看到「序章」首頁，
再從中選擇進入三軌時間軸、或點任一朝代／西方時代查看該時期事件。

## 放上 GitHub Pages（變成可分享的網站）

1. 建立一個新的 repository，把 `index.html` 上傳上去。
2. 進入 repository 的 **Settings → Pages**。
3. Source 選 **Deploy from a branch**，分支選 `main`、資料夾選 `/ (root)`，按 Save。
4. 稍等一兩分鐘，網址會出現在同一頁，形如 `https://你的帳號.github.io/repository名稱/`。

## 檔案說明

- `index.html` — 完整網頁（已內含所有資料、歌曲與程式碼，這是唯一必要的檔案）。
- `序章_列印版.html` — （選用）序章故事頁的列印友善版本。

## 資料來源

時間軸內容為網友整理之輔助複習資料，早期年代（尤其西元前 841 年共和行政以前）
多採傳統說法，學界另有考訂；重要年代仍請對照課本。
